# System Modules for PPT

## 1. Authentication Module
*   **Secure Login**: Role-based access for Admin and Users.
*   **Registration**: New users can create accounts easily.
*   **Session Management**: Protects pages from unauthorized access.

## 2. Admin Module
*   **Dashboard**: Real-time overview of products, low stock, and sales.
*   **Product Management**: Add, update, and delete products with unique QR codes.
*   **Inventory Alerts**: Auto-notification for low stock (< 5) and expiring items.
*   **Sales Reports**: View comprehensive order history and revenue data.

## 3. User Module
*   **Dashboard**: User-friendly landing page.
*   **Scan Product**: Scan QR codes to view product details instantly.
*   **Cart Management**: Add/remove items and adjust quantities.
*   **Purchase History**: Track past orders and view receipts.

## 4. Billing Module
*   **Checkout Process**: Seamless order confirmation and stock deduction.
*   **Invoice Generation**: Auto-generates professional, printable bills.
