# SmartScan Billing System

## Overview

SmartScan Billing System is a QR Code Based Grocery Billing System developed using PHP and MySQL. The application streamlines grocery store billing through product scanning, invoice generation, inventory management, and sales reporting.

## Features

### Admin Module
- Dashboard Analytics
- Product Management
- Inventory Alerts
- Sales Reports
- User Management

### User Module
- User Registration & Login
- Product Scanning
- Shopping Cart
- Checkout Process
- Invoice Generation

## Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap
- WAMP Server

## Installation

1. Install WAMP Server.
2. Copy project folder to:

   C:\wamp64\www\

3. Create a MySQL database:

   smartscan_db

4. Import:

   database.sql

5. Update database credentials in:

   db_connect.php

6. Start Apache and MySQL.

7. Open:

   http://localhost/smart_scan_billing
